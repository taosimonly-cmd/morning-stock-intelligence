Morning Stock Intelligence v3 Cloud Ready

This version is prepared so it can run online. After it is hosted, your phone can open it anywhere using mobile data or any Wi-Fi.

Local test command:
python -m streamlit run app.py

For phone testing at home:
python -m streamlit run app.py --server.address 0.0.0.0

Files included:
- app.py
- requirements.txt
- .streamlit/config.toml
- DEPLOY_ONLINE_STEP_BY_STEP.txt
- APP_STORE_NEXT_STEP.txt
